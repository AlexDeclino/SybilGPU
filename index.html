SYBIL GPU - Help

Text Formatting

  • #N text - Set text size (1-10), e.g., #8 Hello
  • #, ##, ### - Quick sizes (8, 5, 3)
  • [text] - Highlight text with accent color
  • {text} - Full box highlight
  • #N(text) - Inline size change

Images

  • !colId:N - Insert image from library
  • :h - Flip horizontal
  • :v - Flip vertical
  • :c - Keep original colors

Line Modifiers

  • :lhN - Line height (e.g., :lh150 for 150%)
  • :sxN - Stretch X (e.g., :sx200)
  • :syN - Stretch Y
  • :kN - Kerning/letter spacing

Layers

Use L1, L2, L3 tabs to switch between layers. Layer 1 is at the bottom, Layer 3 is on top.

Style Buttons

  • S - Apply size to selection
  • [ ] - Wrap selection in highlight
  • { } - Wrap selection in box
  • ↔ / ↕ - Flip selected image
  • ∗ - Toggle original colors on image
